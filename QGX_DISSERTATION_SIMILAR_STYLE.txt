ABSTRACT
=====================================================================

QGX (Query Gen X) is an AI-enabled Learning Management System designed to digitize
academic workflows and improve learning outcomes through automation, analytics,
and personalized assistance. The platform supports four key stakeholders: Admin,
Teacher, Student, and Parent, each with dedicated dashboards and workflows.

The system is implemented using Next.js 14, React 18, TypeScript, Supabase
(PostgreSQL + Authentication + Row Level Security), and Vercel deployment.
QGX integrates modules for test creation, assignment handling, attendance,
timetable management, messaging, forums, live class support, report cards,
notifications, and student analytics.

A key contribution of the project is AI integration through tutor assistance and
AI-based question generation using GROQ models. In addition, the platform includes
anti-cheat mechanisms for assessment integrity and gamification features such as
XP progression, levels, and leaderboards to increase engagement.

The implementation was validated using unit testing, lint checks, type validation,
and production build testing. Results demonstrate stable deployment readiness,
modular extensibility, and suitability for real-world educational institutions.

Keywords: Learning Management System, AI Tutor, EdTech, Next.js, Supabase,
Gamification, Online Assessment, Role-Based Access Control.

1. INTRODUCTION
=====================================================================

1.1 Purpose of the System

QGX is proposed to provide a unified digital platform for teaching, learning,
assessment, and academic administration. Existing institutional workflows are often
fragmented across spreadsheets, messaging apps, and separate exam portals. QGX aims
to centralize these operations and deliver a consistent user experience.

The system also addresses the need for intelligent learning support through AI
modules and engagement-driven features such as XP progression and achievement-based
motivation.

1.2 Scope of the System

The scope of QGX includes:
- Multi-role authentication and dashboard routing for Admin, Teacher, Student,
  and Parent.
- Test lifecycle: creation, anti-cheat enforcement, attempt, scoring, and analytics.
- Assignment, attendance, timetable, and grade management.
- Real-time messaging, notifications, forum communication, and announcements.
- AI tutor and AI-assisted question generation.
- Parent visibility into attendance and performance metrics.
- Progressive Web App support.

Out of scope for current phase:
- Native mobile application release.
- Camera-based proctoring.
- Institution-level data federation across multiple campuses.

1.3 Current System

Current systems in many institutes suffer from:
- Isolated tools for tests, attendance, communication, and results.
- Delayed feedback loops between teachers, students, and parents.
- Limited analytics and weak personalization.
- Manual effort in evaluation and record maintenance.

1.4 Proposed System

QGX provides a modular, cloud-ready LMS that integrates assessment, communication,
analytics, and AI assistance. The proposed architecture improves operational speed,
transparency, and reliability while maintaining role-based security and scalable
backend design.
