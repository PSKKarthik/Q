ABSTRACT
=====================================================================

QueryGenX is a full-stack AI-enabled Learning Management System (LMS)
designed to improve the digital learning experience for educational institutions.
The system supports four major user roles: Admin, Teacher, Student, and Parent,
with role-specific dashboards and workflows. The platform is developed using
Next.js 14, React 18, TypeScript, Supabase PostgreSQL with Row Level Security,
and deployed on Vercel.

The project integrates multiple academic and administrative modules including test
management, assignment management, attendance tracking, timetable planning,
analytics, notifications, messaging, forum discussions, and report cards. A major
innovation of the project is the AI-based tutoring and AI-assisted question
generation layer using GROQ-based large language models. QueryGenX also introduces
gamification through XP, levels, leaderboards, and activity-based engagement to
improve student participation.

The testing engine supports objective and semi-objective formats such as MCQ,
MSQ, True/False, Fill-in-the-Blank, and Match-the-Following. Built-in anti-cheat
controls such as full-screen checks, tab-switch tracking, question randomization,
and timer control are provided for fair evaluation.

The project is engineered with production readiness in mind. It includes unit
tests, linting, type checking, and end-to-end testing workflows. The final system
achieved stable deployment with all validation checks passing, demonstrating that
QueryGenX is scalable, modular, and suitable for real-time educational environments.

Keywords: Learning Management System, AI Tutor, Next.js, Supabase, Gamification,
Online Assessment, Role-Based Access, EdTech Platform.

CHAPTER 1: INTRODUCTION
=====================================================================

1.1 PURPOSE OF THE SYSTEM

The primary purpose of QGX is to build a smart, scalable, and user-friendly
Learning Management System that digitizes and automates major academic workflows
inside educational institutions. Traditional classroom management and assessment
systems often rely on disconnected tools for attendance, assignments, examinations,
communication, and analytics. QGX addresses this fragmentation by integrating all
these services into one centralized platform.

Another purpose is to improve student engagement using AI and gamification.
Students are more motivated when learning is interactive, feedback is immediate,
and achievements are visible. Therefore, QGX includes XP progression, level-based
rewards, AI tutoring support, and personalized learning modules.

1.2 SCOPE OF THE SYSTEM

The project scope includes:

1. Multi-role access for Admin, Teacher, Student, and Parent.
2. Test and quiz management with AI-assisted question generation.
3. Assignment, attendance, timetable, and course management.
4. Student performance analytics and parent monitoring.
5. Real-time notifications and messaging modules.
6. Forum and collaboration support.
7. PWA capability for mobile-friendly access.

The current scope focuses on web-based deployment. Native Android/iOS apps,
advanced proctoring with camera monitoring, and large-scale analytics forecasting
are considered future extensions.

1.3 CURRENT SYSTEM

In many institutions, academic management is split across multiple systems:
manual records, spreadsheets, messaging apps, and separate exam portals. This
leads to data inconsistency, delayed communication, and limited performance
tracking. Teachers spend extra effort in repetitive administrative tasks, and
students lack unified progress visibility.

1.4 PROPOSED SYSTEM

QGX proposes an integrated cloud-based LMS built on modern web technologies.
The proposed system centralizes all academic operations and provides role-specific
dashboards. Teachers can create tests and assignments, students can attempt tests
with anti-cheat controls, parents can monitor progress, and admins can manage
users and system analytics. AI-powered tutoring and question generation improve
learning quality and reduce content preparation time.
